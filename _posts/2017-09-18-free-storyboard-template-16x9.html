This is a free Storyboards template in PDF format on DIN A4 horizontal (landscape).

This template has 2 frames per sheet each in 16:9 aspect ratio (1.77:1) with accompanying boxes for set plan.

This template is useful if you have slightly complex scenes where you want to indicate where the camera is. The film ratio is 16:9 (1.77:1).

Print it out to draw on paper, the old school way (no cheating).

If you are looking for an editable template similar to this PDF, you can find an InDesign version here: https://gum.co/16x9_sets.
